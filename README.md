# Calculadora Renda Fixa

Calculadora de **juros compostos** para aplicações em renda fixa. Permite simular a evolução do patrimônio com valor inicial, aportes mensais opcionais, taxa de juros (mensal) e prazo em meses.

## Requisitos

- **Node.js** `^20.19.0` ou `>=22.12.0`

## Desenvolvimento

```sh
# Instalar dependências
npm install

# Servidor de desenvolvimento (hot-reload)
npm run dev

# Build de produção (inclui type-check)
npm run build

# Preview do build
npm run preview
```

## Testes e qualidade

```sh
# Testes unitários (Vitest)
npm run test:unit

# Lint (ESLint + oxlint)
npm run lint

# Formatar código (oxfmt)
npm run format
```
